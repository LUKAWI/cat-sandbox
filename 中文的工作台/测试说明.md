### 测试 1：Agent 身份验证
1. 打开网页 http://127.0.0.1:10225
2. 点击左侧 "Turing"（编程专家）
3. 在输入框输入：`你是谁？`
4. 预期回答：`我是 Turing，你的编程专家...`
5. 点击左侧 "Newton"（研究助手）
6. 输入：`你是谁？`
7. 预期回答：`我是 Newton，你的研究助手...`

### 测试 2：对话隔离验证
1. 在 "猫王" 对话框发送：`这是消息 A`
2. 切换到 "Turing" 对话框
3. 验证：**不应该看到** "这是消息 A"
4. 在 "Turing" 对话框发送：`这是消息 B`
5. 切换回 "猫王" 对话框
6. 验证：应该看到 "这是消息 A"，**不应该看到** "这是消息 B"

### 测试 3：多 Agent 对话
1. 依次切换到每个 Agent 并发送消息
2. 验证：每个 Agent 的对话框只显示自己的消息
3. 验证：切换回来时历史消息完整保留
